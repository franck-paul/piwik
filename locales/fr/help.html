Piwik est un logiciel libre (license GPL) de statistiques web. Il produit d'attrayants rapports sur les visiteurs de votre site, les pages les plus consultées, les mots clés de moteur de recherche qu'ils ont utilisés, la langue qu'ils parlent et de nombreuses autres informations.

Rendez vous sur piwik.org pour le télécharger et obtenir d'autres informations.

Piwik doit être installé et fonctionner quelque part afin de pouvoir utiliser ce plugin.

Votre token Piwik
Vous trouverez votre token Piwik en cliquant sur "API" dans l'interface de Piwik. Il s'agit d'un code de 32 lettres et chiffres.

Une fois que vous aurez correctement rempli ces deux champs, vous pourrez choisir un site que vous avez préalablement créé dans Piwik. Si vous devez créer un nouveau site dans Piwik pour votre blog, vous pouvez utiliser le formulaire Créer un nouveau site Piwik pour ce blog.

Vous pouvez éviter que vos propres visites soient comptées en ajoutant une ou plusieurs adresses IP dans le champ Ne pas activer le suivi sur les adresses IP suivantes. Séparez les adresses par des espaces.

Si vous n'aimez pas la façon dont Piwik affiche vos noms de page, vous pouvez cocher Utiliser des noms de page améliorés.
