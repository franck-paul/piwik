Piwik est un logiciel d'analyse web open source (licence GPL). Il fournit des rapports sur les visiteurs de votre site web, vos pages les plus populaires, les mots-clés des moteurs de recherche qu'ils ont utilisés, la langue qu'ils parlent et bien plus encore.

Veuillez consulter le site matomo.org pour le télécharger et obtenir de plus amples informations.

Piwik doit être installé et fonctionnel afin d'utiliser ce plugin.

Votre Token Piwik
Vous trouverez votre jeton Piwik en cliquant sur "API" dans l'interface Piwik. Il se compose de 32 lettres et chiffres.

Une fois que vous avez correctement rempli ces deux champs, vous pourrez choisir un site que vous avez précédemment créé dans Piwik. Si vous avez besoin de créer un nouveau site dans Piwik pour votre blog, vous pouvez utiliser le formulaire Créer un nouveau site Piwik pour ce blog.

Vous pouvez empêcher le suivi de vos propres visites en ajoutant une ou plusieurs adresses IP dans le champ Ne pas suivre les adresses IP suivantes. Il s'agit d'une liste séparée par des espaces.

Si vous n'aimez pas la façon dont Piwik affiche vos noms de pages, vous pouvez vérifier Utiliser des noms de pages fantaisistes.
